# BP 神经网络在线训练收敛方案（精简版）

## 目标

让 `bp_pid_controller.m` 的纯在线学习能稳定收敛，实现基本跟踪。

## 当前架构（不改）

- 3-5-3 网络 + 动量 SGD
- 有限差分 Jacobian
- 增量式 PID + 抗积分饱和
- 每个采样时刻更新一次权值

## 改进方向（4 个抓手，按优先级排）

### 1. Jacobian 估计质量（怀疑是最大瓶颈）

当前有限差分 `dy/du ≈ (y_k-y_{k-1})/(u_{k-1}-u_{k-2})` 在噪声下极不可靠。改进方案：

- **方案 1A**：对 dy 和 du 分别做低通滤波后再求比值
- **方案 1B**：加一个极简的在线辨识器（2-3 个神经元的 RBF 或单层线性网络），输出平滑的 dy/du
- **方案 1C**：回归简单——直接假设 `dJ_du = -e * sign(Δy)`（放弃估计幅值，只保留方向），在工厂增益方向已知时这是最鲁棒的做法

### 2. 学习率策略

当前固定 lr=0.05，无衰减。改进方案：

- **方案 2A**：指数衰减 `lr = lr0 * exp(-t/τ)`
- **方案 2B**：自适应——连续 N 步误差增大则减半 lr，误差减小则缓慢恢复
- **方案 2C**：两阶段——前 T 秒用较大 lr 快速探索，之后切换到小 lr 精细调节

### 3. 打破样本相关性

连续两步的输入几乎相同，相当于重复学习同一个样本。改进方案：

- **方案 3A**：每 k 步更新一次（如 k=5），降低更新频率，提高单次更新时样本的差异性
- **方案 3B**：mini-batch——攒够 N 个样本取平均梯度后再更新

### 4. 冷启动缓解

随机权重的初始控制量可能让系统几步之内就飞了。改进方案：

- **方案 4A**：输出层偏置初始化为非零值，让初始 Kp/Ki/Kd 落在合理范围（而不是 sigmoid 中点对应的中间值）
- **方案 4B**：前 100 步冻结 BP 权重，用固定 PID 跑，同时只收集数据（不做在线更新），100 步后再开启在线学习

## 实验设计

- **被控对象**：先用最简单的 `10/(s²+2s+5)`（二阶线性），先排除对象复杂度的影响
- **参考信号**：阶跃信号（单一、可预期）
- **收敛判据**：
  - 定性：输出 y 能否最终跟踪上 r（看 Scope 图）
  - 定量：最后 500s 的 RMS 误差 < 0.05 算基本收敛

## 实施策略

一次只改一个变量。先单独测每个改进独立的效果，找到有效的再组合。
